# Tencent Ad Demo

影游共生互动广告原型。项目以腾讯视频风格的主视频播放器为入口，在主视频结束后进入角色选择与互动小游戏流程，并在结算页展示广告权益、优惠券、会员权益和角色加分反馈。

## 入口

直接打开：

```text
index.html
```

## 主要内容

- 主视频播放器与角色选择流程：`index.html`
- 备用完整角色流程页：`character-selection-dialogue-new.html`
- 京东商品购买 UI：`jd_product.html`
- 小游戏快照目录：`snapshots/`

## 小游戏

- `snapshots/1马车游戏/realescort_game.html`
- `snapshots/2密库寻宝/treasure_hunt.html`
- `snapshots/3锦盒连开/index.html`
- `snapshots/4监察院密档/qingyunian-puzzle-game.html`
- `snapshots/5时空议价/qingyunian-game.html`

## 说明

这是本地交互原型，页面中的购买、会员、优惠券等能力均为演示 UI，不会发起真实购买或真实权益领取。
